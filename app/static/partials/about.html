Data Scraping

Our data is being pulled from the Giant Bomb API. The Giant Bomb API is a RESTful API that responds to HTTP GET requests. They have a very wide variety of information over lots of video game related topics so we were able to get creative with our pillar selection. We ended up choosing the pillars that we did because we thought it would be interesting, useful, and different than any other group's topics. To get an API key we made an account and made an agreement to credit them in the work that we do. Once we had this key we were able to generate our JSON files by simply creating a variable in a .py file and executing curl commands on the command line to generate the JSON files. In order to acquire each JSON file we simply changed the URL of the game/platform/character we were trying to access.
